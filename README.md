# Personal Portfolio

A responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. This project showcases my background, technical skills, education, and featured projects through a clean, modern, and user-friendly interface. Portfolio content is dynamically loaded from a JSON file, making it easy to update and maintain without modifying the HTML structure.

## Features

- Responsive design for mobile, tablet, and desktop
- Dynamic content loaded from JSON
- Skills, projects, and education/experiences timeline
- GitHub and Figma project links
- Project preview images
- Interactive navigation and scroll progress indicator
- Clean and organized user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- JSON

## Purpose

This portfolio serves as a personal showcase of my web development journey, highlighting the projects, skills, and experiences I have gained while learning programming.
